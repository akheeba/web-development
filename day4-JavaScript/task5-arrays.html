Day 4 - JavaScript Arrays

Introduction

An array is a special variable that can store multiple values in a single variable. Each value in an array is called an element.

Array Methods Demonstrated

Method Purpose
push() Adds an element to the end of the array.
pop() Removes the last element from the array.
shift() Removes the first element from the array.
unshift() Adds an element to the beginning of the array.
length Returns the total number of elements in the array.

Initial Array

["Apple", "Banana", "Mango", "Orange", "Grapes"]

Expected Output

Original Array:
["Apple", "Banana", "Mango", "Orange", "Grapes"]

After push():
["Apple", "Banana", "Mango", "Orange", "Grapes", "Pineapple"]

After pop():
["Apple", "Banana", "Mango", "Orange", "Grapes"]

After shift():
["Banana", "Mango", "Orange", "Grapes"]

After unshift():
["Watermelon", "Banana", "Mango", "Orange", "Grapes"]

Array Length:
5

How to View the Output

Open the browser console (F12 → Console) to see how the array changes after each operation.
