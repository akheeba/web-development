Day 4 - JavaScript User Input using prompt()

Introduction

The prompt() function displays a dialog box that allows the user to enter input. By default, the input received is of type string. Therefore, we convert it into a number before performing arithmetic operations.

Functions Used

Function Purpose
prompt() Accepts input from the user.
Number() Converts a string into a number.
console.log() Displays output in the browser console.

Program Steps

  1. Ask the user to enter the first number.
  2. Ask the user to enter the second number.
  3. Convert both inputs to numbers.
  4. Perform arithmetic operations.
  5. Display the results in the console.

Expected Output

Enter First Number: 20
Enter Second Number: 5

Addition: 25
Subtraction: 15
Multiplication: 100
Division: 4

Open this page in your browser. Two prompt boxes will appear automatically. Press F12 → Console to view the results.
