Day 4 - JavaScript Primitive Data Types

Introduction

A data type defines the kind of value that a variable can store. JavaScript has 7 primitive data types, each designed to represent different kinds of information.

The 7 Primitive Data Types

Data Type Example Description
String "Hello" Stores text.
Number 25 Stores integers and decimal numbers.
Boolean true Stores either true or false.
Undefined let x; A variable declared without a value.
Null null Represents an intentional empty value.
BigInt 123456789n Stores very large integers.
Symbol Symbol("id") Creates a unique identifier.

typeof Operator

The typeof operator returns the data type of a variable or value.

Example:

typeof "Hello"      → string
typeof 20           → number
typeof true         → boolean
