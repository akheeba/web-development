Day 4 - JavaScript Comparison Operators

Introduction

Comparison operators compare two values and return either true or false.

This practical demonstrates the difference between == (Loose Equality) and === (Strict Equality).

Difference

Operator Meaning
== Compares only the values after type conversion if necessary.
=== Compares both value and data type without type conversion.

Examples Used

"10" == 10
"10" === 10

true == 1
true === 1

null == undefined
null === undefined

Expected Result

"10" == 10          → true
"10" === 10         → false

true == 1           → true
true === 1          → false

null == undefined   → true
null === undefined  → false

Open the browser console (F12 → Console) to view the output.
