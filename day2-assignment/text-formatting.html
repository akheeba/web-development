HTML Text Formatting Tags

<strong> - Defines text with strong importance. It is usually displayed in bold.

Example: This is strong text.

<b> - Makes text bold without adding semantic importance.

Example: This is bold text.

<em> - Emphasizes text. It is usually displayed in italics.

Example: This is emphasized text.

<i> - Displays text in italics without adding emphasis.

Example: This is italic text.

<mark> - Highlights text.

Example: This text is highlighted.

<small> - Displays text in a smaller font size.

Example: This is small text.

<sub> - Displays subscript text.

Example: H2O

<sup> - Displays superscript text.

Example: x2

<u> - Underlines text.

Example: This text is underlined.

<del> - Represents deleted text.

Example: ₹500

<ins> - Represents inserted text.

Example: ₹400

<code> - Displays inline computer code.

Example: print("Hello World")

<kbd> - Represents keyboard input.

Example: Press Ctrl + S to save.

<q> - Defines a short quotation.

Example: Learning never stops.

<blockquote> - Defines a long quotation.

The beautiful thing about learning is that no one can take it away from you.

<abbr> - Defines an abbreviation.

Example: HTML

<cite> - Defines the title of a creative work.

Example: Harry Potter

<br> - Inserts a line break.

Example:
This is the next line.

<hr> - Inserts a horizontal line.
