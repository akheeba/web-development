Day 5 - Arrow Functions

Introduction

Arrow functions were introduced in ES6 (ECMAScript 2015). They provide a shorter and cleaner way to write functions.

Traditional Function vs Arrow Function

Traditional Function Arrow Function 
function greet(name) {
    return "Hello " + name;
}

const greet = (name) => {
    return "Hello " + name;
};

Concepts Demonstrated

  • Traditional Functions
  • Arrow Functions
  • Parameters
  • Return Statement
  • Implicit Return
