Day 5 - let, const and var

Introduction

JavaScript provides three keywords to declare variables: var, let, and const. Each has different rules regarding scope and reassignment.

Comparison

Keyword Can Reassign? Scope
var Yes Function Scope
let Yes Block Scope
const No Block Scope

Concepts Demonstrated

  • Variable declaration
  • Reassignment
  • Block Scope
  • Function Scope

Expected Output

let value: 20

const PI: 3.14159

var inside block: I am var
var outside block: I am var

let inside block: I am let
let outside block: Error (not accessible)

Open the browser console (F12 → Console) to view the output.
