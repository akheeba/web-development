Day 5 - Spread Operator (...)

Introduction

The spread operator (...) was introduced in ES6. It expands the elements of an array or the properties of an object into another array or object.

Uses of the Spread Operator

Use Description
Copy an Array Creates a new copy of an existing array.
Merge Arrays Combines multiple arrays into one.
Merge Objects Combines properties of multiple objects.

Concepts Demonstrated

  • Copying an Array
  • Merging Arrays
  • Merging Objects
  • ES6 Spread Operator (...)

Expected Output

Original Array:
["Apple", "Banana", "Mango"]

Copied Array:
["Apple", "Banana", "Mango"]

Merged Array:
["Apple", "Banana", "Mango", "Orange", "Grapes"]

Merged Object:
{
  name: "Akheeba",
  age: 18,
  course: "ECE",
  city: "Nellore"
}

Open the browser console (F12 → Console) to view the output.
